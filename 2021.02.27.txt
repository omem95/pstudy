1. HTML이란?
HTML의 HT = HyperText(링크형식의 시스템), M = Markup(구조를 설정할 때 사용), L = Language(언어)
>> 구조를 설계할 때 사용되는 언어로 하이퍼링크 시스템 활용

HTML의 확장자는 HTM(L)

HTML은 단순한 텍스트 파일인데 웹브라우저가 해석해서 화면에 렌더링해줌


2.시멘틱 마크업
txt파일과 다르게 html파일은 웹브라우저가 해석을 해서 구조를 설계함

이 구조설계는 웹브라우저에서 우클릭 -> 검사 -> Elements에서 확인 가능

사람이 작성한 컨텐츠를 브라우저가 해석하며 자동으로 설계를 구성하는 것을 마크업이라고 함

HTML은 웹사이트 콘텐츠를 설명하는데 사용되는 마크업 언어로 콘텐츠의 의미 설명이 유일한 목적

HTML엔지니어는 브라우저가 올바르게 해석할 수 있도록 하는 것이 목표

HTML의 요소는 태그로 시작과 끝을 감싸줘야함


3. 기본 문법
웹페이지는 헤드 영역, 바디 영역으로 구성됨

자주 사용되는 용어
elements 요소
open tag 여는 태그
close tag 닫는 태그
attribute 속성
value 값

기본 문법 : <태그이름 속성="값"> 컨텐츠 </태그이름> ->태그 처음부터 끝까지가 요소

브라우저에서 요소에 커서를 올렸을 때 작게 표시되는 창을 툴팁이라고 함, title 속성 값이 출력됨

코드 중 사용자가 보는 것은 body 내부, 브라우저에 정보를 주는 것은 head 내부

html태그는 내부 컨텐츠의 뿌리가 되기 때문에 root elements라고 함

글자가 깨져보인다면 글자 해석(인코딩)에 실패한 것

브라우저의 인코딩 확인 방법 : 우클릭 -> 검사 -> Console -> document.characterSet입력

인코딩 설정 방법 : 헤드태그 내부에 <meta charset="utf-8">
